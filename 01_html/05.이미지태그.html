이미지태그 img

이미지태그는 이미지를 연결시켜 문서에 보이도록 해주는 태그

img 태그는 닫는 태그 없음

링크 연결할 때는 href 속성 사용, img태그에 사용하는 속성은 scr

alt 라는 속성으로 사용된 이미지에 대한 설명을 입력 합니다

위에서 사용된 alt 속성은 이미지가 깨지거나, 엑박이 뜨더라도 텍스트로 표현 됨

이미지가 잘 나온다면 보이지 않습니다

img 태그는 인라인 요소 입니다.





작성중인 html 파일과 동일한 위치에 이미지를 불러올 경우: ./파일이름.확장자


작성중인 html 파일과 동일한 위치에 폴더가 있고, 그 안에 있는 이미지를 불러올 경우: ./폴더이름/파일이름.확장자


작성중인 html파일과 동일한 위치에 폴더가 있고, 그 안에있는 폴더에 안에있는 이미지를 불러올 경우 : ./폴더이름/폴더이름/파일이름.확장자


작성중인 html 파일보다 더 상위에 폴더가 있고, 그 안에있는 이미지를 불러올 경우 : ../파일이름.확장자
