table

  1. table 태그는 표를 만들 때 사용하는 태그
  2. 옛날 홈페이지를 제작하는 방식인데 수정이 어렵고, 잘 일그러져서 수정하고 확인하고를 반복해야 한다
  3. 행(row)과 열(Column)로 구성된다. 오-가로와 열-세로, 행-가로 종-세로
  4. table -> tr -> tr/td 의 순서로 마크업 한다 tr, td, th는 모두 자식 요소로 들어오는 태그
  5. table에 속성값으 추가해서 선을 만들고 (border="선 굵기"), 선의 간격 (cellspacing="간격 수치") 과 셀의 여백 (cellpadding="간격 수치") 과 표의 넓이 (width="너비값"), 표의 높이 (height=높이값") 을 조절 할 수 있다
  6. tr태그에 align 속성으로 내용의 글자를 정리
  7. td: 열(Column)을 정의할 때 사용하는 태그
  8. th : td와 같은 기능, 제목역할을 하는 셀을 정의할 때 사용
