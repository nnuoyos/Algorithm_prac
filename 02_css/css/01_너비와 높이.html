width : 요소의 너비를 나타내는 속성

  • 단위: px, %, auto(정해진 너비를 없앨 때 사용)
  • 300px
  • 100px
  • 50%

height : 요소의 높이

  • 단위: px, %, auto(정해진 너비를 없앨 때 사용)
  • 250*200
  • 200px

min-width: 요소의 최소 너비

단위: px, %, auto(정해진 너비를 없앨 때 사용)

최소 너비800px

max-width: 요소의 최대 너비

단위: px, %, auto(정해진 너비를 없앨 때 사용)

최대 너비800px

min-height: 요소의 최소 너비

단위: px, %, auto(정해진 너비를 없앨 때 사용)

max-height: 요소의 최대 너비

단위: px, %, auto(정해진 너비를 없앨 때 사용)
