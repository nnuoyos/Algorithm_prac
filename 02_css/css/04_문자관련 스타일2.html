문자관련 스타일2

  1. word-spacing: 단어 사이의 간격 조정

    단위: px, em

    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기 위해 사용자의 요구사항을 분석. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 디자인을 검토한다 블라블라

  2. font-style : 글자의 표현 방식

    속성값: normal, italic, oblique

    글자의 표현 방식 나타내기

    adress태그: 주소, 사업자등록번호 등등을 작성하는 시멘틱태그. 글자 기울임이 기본값으로 적용되어 있음


  3. text-decoration: 텍스트 장식

    속성값: underline, overline, line-through, none

    텍스트에 줄 긋기

    링크태그, 밑줄과 글자색이 기본으로 들어온다 (데코레이션으로 색상과 밑줄 없앨 수 있음)

  4. text-indent: 텍스트 들여쓰기

    단위: px


    휴대폰 UI/UX 기획자 휴대폰의 편리한 사용자 인터페이스를 제공하기 위해 사용자의 요구사항을 분석. 하드웨어 특성을 고려한 UI/UX의 기능을 분석하여 디자인을 검토한다 블라블라



  5. text-shadow: 글자 그림자

    text-shadow: (offset-x)(x축 수치)
    px(offset-y)(y축 수치)
    (blur)px 색상;



    글자 그림자는 페이지크기에 영향을 주지 않는다

    blur 수치는 음수는 안되고, 나머지 통상 수치가 음수가 되면 줄어들거나 거꾸로 되는 느낌으로 나타난다

    rgba값으로 색상을 설정할 수 있다

    rgba값으로 그림자 주기
