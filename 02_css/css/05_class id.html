class

태그에 이름을 줄 때 사용, 여러개의 태그에 같은 이름을 사용할 수 있다.
띄워쓰기 하나로 하나의 태그에 여러개의 이름도 사용 가능

ID

태그에 고유명칭을 줄 때 사용. 여러개의 태그에 같은 이름을 사용할 수 없다
아이디와 클래스는 같은 이름으로 사용 가능

규칙

클래스나 아이디의 이름은 영어로 작성. 대소문자를 구분. 시작자리에 들어갈 수 있는 기호는 없다.
기호는 영문을 사용한 다음, 중간에 _(언더바), ~(물결)을 사용가능
관례적으로 내용과 관련이 있는 영문이름으로 사용

(내부 스타일시트의 경우)헤드태그의 타이틀태그 아래에 스타일태그 안에서 사용되어 진다
중괄호{}안에 들어갈 속성들을 작성한다

id는 #으로 호출, class는 . 으로 호출한다 나중에 적은 값이 우선순위로 적용된다
클래스보다 아이디가 우선 적용된다

    클래스나 아이디의 이름을 작성하는 방법
  • Sbs academyfrontend : 예시로 사용될 이름
  • Camel Case : SbsAcademyFrontEnd
  • Snake Case : sbs_academy_frontend
