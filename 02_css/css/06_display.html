display

속성값 : block, inline, inline-block, none



display : block

인라인 요소를 블락 요소로 바꾸어 준다.

box1 box2 box3 box4
box1 box2 box3 box4


display: inline

블락요소를 인라인 요소로 바꾸어 준다.

box1

box2

box3

box4



display : inline-block

인라인요소나 블락요소를 인라인블락요소로 바꾸어 준다.
인라인요소를 가지고 있기 때문에 줄바뀜이 일어나지 않는다
그러나 너비,높이,text-align 등등 블락요소에만 적용되는 속성들을 사용할 수 있게 된다

inline-block 값을 준 요소의 상위태그에 text-align값을 주면 나열된 요소의 그룹들이 정렬된다
