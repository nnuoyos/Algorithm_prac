border : 요소의 테두리를 지정

border: 선두께 선스타일 선색상;

너비와 높이에 영향을 준다.

border-top / border-bottom / border-left / border-right

border-width: 2px; >> 테두리 선의 두께
border-style: solid, dotted, dashed, none, double, hidden, groove, ridge, inset, outset
border-color: 색상명, 색상코드, rgb, rgba

선이 있을때 margin을 쓰면 선 밖으로 여백이 생기고, padding을 쓰면 선 안으로 여백이 생김.
