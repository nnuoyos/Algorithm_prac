link 태그로 연결해서 사용

html파일에 사용

  1. 구글폰트 검색 > 한국어
  2. 원하는 글자체 선택
  3. + select this style
  4. link부분 복사 후 head태그 안에 붙여넣기
  5. CSS rules to families부분 복사해서 해드>스타일태그 안에 적용할 태그에 넣기


@import로 연결해서 사용

스타일시트에 사용(외부, 내부 둘다 가능)

  1. 구글폰트 검색 > 한국어
  2. 원하는 글자체 선택
  3. + select this style
  4. @import부분을 복사한 후 스타일시트 안에 붙여넣기
  5. CSS rules to families부분 복사해서 적용할 태그에 넣기
