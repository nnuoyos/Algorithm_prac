말줄임

말줄임을 하기 위해서는 고정 너비가 필요


한 줄 일때 말줄임

  1. overflow: hidden; :고정 너비를 벗어난 컨텐츠를 숨김
  2. white-space: nowrap; :자동 줄바꿈 방지
  3. text-overflow : ellipsis; :너비를 벗어난 컨텐츠를 '...'로 표시

여러줄일때 말줄임

  1. overflow: hidden;
  2. display : -webkit-box;
  3. -webkit-box-orient : vertical; :방향설정
  4. -webkit-box-clamp: 줄수(숫자)

-webkit 브라우저(크롬,오페라,사파리)의 여러줄 말줄임을 위한 속성

그 외 브라우저는 line-height와 height를 설정해주어야 함
