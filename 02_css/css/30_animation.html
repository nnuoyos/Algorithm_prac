animation : @keyframes 를 이용해 애니메이션을 만든다

  1. animation-name : @keyframes 이름;
  2. animation-duration : 애니메이션 진행 시간 (s)
  3. animation-timing-function: 애니메이션 속도변화 유형;
    • ease : 느린시작으로 빨리진 뒤 느린 끝
    • linear : 처음부터 끝까지 일정한 속도
    • ease-in: 느린시작
    • ease-out: 느린끝
    • ease-in-out:느린시작과 끝
  4. animation-delay : 애니메이션 시작하기 전 지연시간
  5. animation-iteration-count: 애니메이션 재생횟수를 지정(N, infinite 무한대)
  6. animation-direction : 애니메이션 종료 후 반복될 때 진행하는 방향
    • normal: 기본값, 0% > 100%
    • reverse: 역방향, 100%>0%
    • alternate: 정방향으로 움직이고 역방향으로 다시 감, 0%>100%>0%
    • alternate-reverse: 역방향으로 움직이고 정방향으로 다시 감, 100%>0%>100%
    • 단, alternatesms interation-count를 2이상으로 해야 정,역방향 모두 볼 수 있음
  7. animation-fill-mode: 애니매이션 미실행시(시작할때말고, 종료/대기) 요소의 스타일
    • none : 기본값, 애니메이션 끝난후 상태 지정 않음 그냥 끝
    • forwards : 끝난 후 그 지점 그대로 있음 끝난 상태 그대로 끝남
    • backwards : 끝난 후 시작점으로 돌아옴
    • both : 앞뒤 결과를 조합해서 설정
    • inherit : 상위요소에 상속
  8. animation-paly-state : 애니메이션 재생상태(재생/중지)를 지정
  9. animation:(name)(duration)(timing-function)(delay)(iteration-count)(direction); 괄호는 구분을 위해 쓴 것
