추상화

  • 추상이란 여러가지 사물이나 개념에서 공통되는 특성이나 속성 따위를 추출하여 파악하는 작용을 한다
  • 구체적인 정보는 숨기고 꼭 필요한 핵심만을 설명하는 것을 추상화라고 한다 Abstranction
  • ex) 책 제목, 영화의줄거리 -> 세부적인 스토리나 전개, 배경은 숨기고 핵심만을 말해줌
    추상화의 결과라고 볼 수 있음
  • 추상화를 위해서는
    목적을 명확하게 할 것
    불필요한 것은 숨길 것
    핵심만을 드러낼 것

변수

  • 데이터를 담아두는 상자의 역할
  • 변하는 수를 담을 수 있다고 해서 변수라고 한다
  • 영어로는 Variable
  • 하나의 변수에는 하나의 데이터만 담을 수 있다
  • 새로운 값이 지정되기 위해서는 원래 저장되어 있던 값은 버려지게 된다
  • 변수를 만드는 방법(변수를 선언하는 방법)은 다음과 같다
      - var 변수이름 = 변수에 저장할 내용
      - let 변수이름 = 변수에 저장할 내용
       - 통상적으로 let을 많이 사용한다
      - const 변수이름 = 변수에 저장할 내용(이 경우 상수의 저장에 사용된다)
       *상수는 처음 선언할 때 값을 무조건 지정해야 함
       * 상수는 변하지 않는 고정값
       * 상수의 이름은 대문자로 선언(오류는 일어나지 않지만 개발자들 사이에서의 관례)
  • 변수 선언 시 값을 지정하지 않아도 된다
  • 변수의 이름에는 절대로 따옴표를 사용하면 안됟나. 따옴표를 사용하다는 것은 문자형 자료를 사용한다는 의미이므로 결과값이 완전히 달라진다

변수 이름 만드는 규칙

  • 시작은 반드시 영문 알파벳 언더바_, 달러기호($) 로만 시작해야 한다
    두번째 글자부터는 숫자도 사용 가능
  • 대소문자는 서로 다른 대상으로 인식 된다
  • 예약어(키워드, JavaScript가 먼저 의미부여 해놓은 단어들)는 사용 불가능
  • 그 외 지키면 좋은 규칙들
      - 의미없는 이름 사용하지 않기
      - 너무 추상적인 이름은 좋지 않다
      - 모든 변수의 이름은 'camelCase', 'snake_case', 파스칼(NameRule)을 사용하여 작성하면 좋다
